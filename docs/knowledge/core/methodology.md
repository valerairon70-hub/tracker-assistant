# Метод Бутаковой — методология

## Что такое трекер здоровья

120 вопросов (10 на каждую из 12 систем) → диаграмма → протокол восстановления.
Это НЕ медицинская диагностика. Работаем с СОСТОЯНИЕМ, не с болезнями и диагнозами.
Никогда не использовать слова: "лечение", "болезнь", "диагноз" — только "состояние", "нагрузка", "перегруз", "восстановление".

## Цепь развития проблем

Состояние (функциональный сбой) → Жалобы → Симптомы → Синдром → Диагноз → Болезнь

Продукты Coral Club работают на уровне «состояние» — до болезни, в её профилактике и параллельно с медикаментами.

## 7 шагов восстановления (ОБЯЗАТЕЛЬНЫЙ ПОРЯДОК)

1. **Психологический тренинг** — убрать деструктивные программы (страх, обиды, стресс)
2. **Питание** — исключить вредное, добавить необходимое по системам
3. **Вода** — норма 30 мл/кг, структурированная (Корал-майн)
4. **Клеточное питание (АМК)** — аминокислоты + минералы + коллаген; продукт: Противити Ультра
5. **Системная очистка** — строго по порядку: ЖКТ → сосуды → лимфа → клетка
6. **Восстановление органа** — специфические протоколы по нагруженным системам
7. **Поддержка** — 90-дневный цикл, затем повторный трекер

**Этот порядок критичен.** Нарушение этапности снижает эффект. Не начинать восстановление органа до очистки.

## Системная очистка — строгий порядок (Шаг 5)

1. ЖКТ — пробиотики (Би-Курунга, Супер-Флора) + ферменты (Ассимилятор, Папайя)
2. Сосуды — Омега-3 TG + ЦиркуФит + Коэнзим Q10
3. Лимфа — ЛимФлоу + Корал-Майн + H-500
4. Клетка — АМК + витамины группы B + Омега-3

## Клеточное питание (АМК) — Шаг 4

АМК = Аминокислоты + Минералы + Коллаген
Продукт-основа: **Противити Ультра** (11 незаменимых аминокислот)
Клетка не восстановится без строительного материала — никакой орган не заработает нормально.
Порядок: сначала АМК, потом специфические протоколы по системам.

## 12 жизненных констант (Точка А — объективные маркеры)

- АД 120/80 мм рт.ст.
- ЧСС 78 уд/мин
- ЧДД 16 вдохов/мин
- Гемоглобин 130 г/л
- рН крови 7.43
- Сахар 5.5 ммоль/л
- Температура 36.6°С
- Лейкоциты 4000 ед/мкл
- Холестерин 6.5 ммоль/л
- Билирубин 20 мкмоль/л
- Суточная моча 1.5 л
- Вес = (Рост − 100) ± 10 кг

## 12 факторов болезни (колесо)

Психология, Еда, Вода, АБВГДИП (витамины/минералы), Медикаменты, Отсутствие движения, Вредные привычки, Наследственность, Травмы, Биоэнергетика, Экология, Время.

Использовать как чек-лист при анализе: почему система в перегрузе.

## Цвета диаграммы

🟢 ЗЕЛЁНЫЙ = НОРМА — 0–4 «да» (0–40%) — система работает оптимально
🟡 ЖЁЛТЫЙ  = НАГРУЗКА — 5–7 «да» (50–70%) — первые сигналы, обратимо, нужна коррекция
🔴 КРАСНЫЙ  = ПЕРЕГРУЗ — 8–10 «да» (80–100%) — предболезнь, истощена, срочная коррекция

Стандартный курс восстановления — 90 дней, затем повторный трекер.

## Интерпретация текстовых данных трекера

- Число 1–10 → количество «да»-ответов. Умножь на 10 чтобы получить %.
  Пример: «ЦНС — 8» = 80% = 🔴 КРАСНЫЙ
- Число 11–100 → уже процент.
  Пример: «ЦНС — 80%» = 80% = 🔴 КРАСНЫЙ
- «Нет баллов» / «0» → 0% = 🟢 ЗЕЛЁНЫЙ

## Правило подбора по бюджету

- **Нет бюджета / первое знакомство** → Уровень 1: 1 продукт для самой нагруженной системы
- **Начальный бюджет** → Уровень 2–3: синергия уже работает
- **Комфортный бюджет** → Уровень 4: полный протокол на 2–3 приоритетных системы
- **Хочет «всё и сразу»** → Уровень 5: ПЭК-комплекс, удобно и системно
