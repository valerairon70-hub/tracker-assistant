# ЦНС — Центральная нервная система

## Программы по жалобам

**Бессонница / плохой сон**
→ "Программа от бессонницы": Гриффония + Корал Таурин + Корал Магний + Ивнинг Формула

**Стресс (мужчина)**
→ "Антистресс для мужчин": Гриффония + Сафрино + Мемо-Прайм + Ивнинг Формула

**Стресс (женщина)**
→ "Противострессовая для женщин": Гриффония + Сафрино + Мемо-Прайм + Ивнинг Формула

**Хроническая усталость / нет энергии**
→ "От хронической усталости": Коэнзим Q10 + D-Спрей 2000 + Алтимейт Макс + Омега-3 TG

**Плохое настроение (мужчина)**
→ "Стабилизация настроения для мужчин": Гриффония + Корал Магний + Омега-3 TG + Тейсти Би

**Плохое настроение (женщина)**
→ "Стабилизация настроения для женщин": Гриффония + Корал Магний + Омега-3 TG + Тейсти Би

**Депрессия / тревога**
→ "От депрессии и тревоги": Гриффония + Ивнинг Формула + Омега-3 TG + Би-Прайм Директ
(при серьёзных симптомах — метилированные формы работают быстрее; при лёгкой форме допустим Тейсти Би)

**Головные боли / мигрень**
→ "От головных болей и мигрени": Готу Кола + Корал Магний + Омега-3 TG + МайндСет

**Память / концентрация**
→ "Улучшение памяти и внимания": Лецитин Подсолнечный + Корал Магний + Мемо-Прайм

**Головокружение / шум в ушах**
→ "От головокружения и шума в ушах": Корал Магний + Готу Кола + Лецитин + МайндСет

**Возраст 60+**
→ "ЦНС 60+": Коэнзим Q10 + Цинк Метионин 25 + Корал Магний + Омега-3 TG

## Питание для ЦНС
- Убрать: алкоголь, кофе >2 чашек/день, трансжиры, рафинированный сахар
- Добавить: магний (тыквенные семечки, орехи), триптофан (бананы, индейка), жирная рыба, B-витамины (яйца, злаки)

## Практики для ЦНС
- Режим сна: ложиться до 23:00, темнота, прохладный воздух; подъём в одно время
- Дыхание: 5 мин утром — 4 сек вдох, 6 сек выдох (снижает кортизол)
- Детокс: цифровой детокс за 1 час до сна; прогулки 20–30 мин/день
