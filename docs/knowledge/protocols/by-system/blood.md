# КС — Кроветворная система

## Программы по жалобам

**Ферритин у детей / детская анемия**
→ "Клеточное питание для повышения ферритина у детей": Спирулина Премиум + DHA+D3 Smart Chews + Би-Курунга + Ямми Витс

**Ферритин у спортсменов / усталость после нагрузок**
→ "Клеточное питание для повышения ферритина у спортсменов": Спирулина Премиум + D-Спрей 2000 + H-500 + Алтимейт Макс

**Беременность / кормление**
→ "Клеточное питание для повышения ферритина у беременных": Спирулина Премиум + Железо + Пренатал+ + Лецитин

**Высокий ферритин / воспаление / печень**
→ "Очищение печени для понижения ферритина при воспалениях": Заферан + Корал Артишок + Джинерра + Корал Бурдок Рут

**Гемоглобин у женщин / физиологические состояния**
→ "Поддержание гемоглобина при физиологических состояниях у женщин": Спирулина Премиум + D-Спрей 2000 + Алтимейт Макс + Селен

**Анемия / низкий гемоглобин**
→ "Защита крови и восстановление при анемии": Железо + Би-Прайм Директ + Хлорелла + Селен
(метилкобаламин + метилфолат критичны при анемии — заменяют Тейсти Би)

**Низкий ферритин**
→ "Повышение и стабилизация ферритина": Спирулина Премиум + D-Спрей 2000 + Тейсти Би + Коэнзим Q10

## 5-уровневая эскалация для КС

| Уровень | Продукты |
|---------|---------|
| 1 | Спирулина Премиум |
| 2 | Спирулина Премиум + Железо |
| 3 | Спирулина Премиум + Железо + Витамин D2000 |
| 4 | Спирулина Премиум + Железо + Витамин D2000 + Pure-C |
| 5 (ПЭК) | Нутрипэк |

## Важные ограничения
- Железо (Ивнинг Формула, Железо) у мужчин и женщин 50+ — только при подтверждённом дефиците ферритина (<30 мкг/л)
- Тейсти Би (B9) без B12 при анемии → добавлять B12 или проверять его статус

## Питание для КС
- Убрать: кофе/чай сразу после еды (блокируют усвоение железа), алкоголь
- Добавить: красное мясо/субпродукты, бобовые, шпинат, витамин С для усвоения железа
