# ЭС — Эндокринная система

## Программы по жалобам

**Лишний вес + аппетит/стресс**
→ "Снижение веса при повышенном аппетите": Корал Карнитин + Липостик Фит + D-Спрей 2000 + Корал Магний

**Лишний вес + тяга к сладкому**
→ "Снижение веса при инсулинорезистентности": Корал Карнитин + Слим Бай Слим + D-Спрей 2000 + Корал Магний

**Поддержание веса / удержать результат**
→ "Поддержание веса": Дейли Делишес + Гриффония + Корал Люцерна

**Щитовидная железа / гипотиреоз**
→ "Поддержка щитовидной железы при гипотиреозе": Корал Келп + КМД с Селеном + D-Спрей 2000 + Сильвер-Макс
⚠️ Корал Келп — только с разрешения эндокринолога. ТТГ обязателен.

**Уровень сахара / диабет риск**
→ "Стабилизация уровня сахара": Цинк Метионин 25 + Корал Магний + Корал Бурдок Рут + Заферан

**Надпочечники / хронический стресс**
→ "Укрепление надпочечников": Тейсти Би + Противити Ультра + Сафрино + Корал Магний

## 5-уровневая эскалация для ЭС

| Уровень | Продукты |
|---------|---------|
| 1 | Липостик Фит |
| 2 | Липостик Фит + Тейсти Би |
| 3 | Липостик Фит + Тейсти Би + Корал Келп |
| 4 | Липостик Фит + Тейсти Би + Корал Келп + Корал Карнитин |
| 5 | Липостик Фит (базовый) |

## Питание для ЭС
- Убрать: сою при гипотиреозе, сырую капусту/брокколи при гипотиреозе (блокируют йод), сахар, алкоголь
- Добавить: морские водоросли, йодированную соль, селен (бразильский орех), хром (цельнозерновые)
