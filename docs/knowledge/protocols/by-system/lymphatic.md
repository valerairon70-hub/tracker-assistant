# ЛС — Лимфатическая система

## Программы по жалобам

**Общая интоксикация**
→ "Очищение лимфы": ЛимФлоу + Корал Люцерна + Корал Солодка + Цинк Метионин 25

**Отёки / задержка жидкости**
→ "Программа от отёков": МСМ + ЦиркуФит + Корал Бурдок Рут + Корал Таурин

**Воспалённые лимфоузлы**
→ "Восстановление лимфоузлов при воспалениях": Цинк Метионин 25 + Корал Солодка + Корал Кошачий Коготь + КМД с мате

## 5-уровневая эскалация для ЛС

| Уровень | Продукты |
|---------|---------|
| 1 | ЛимФлоу |
| 2 | ЛимФлоу + H-500 |
| 3 | ЛимФлоу + H-500 + Корал-Майн |
| 4 | ЛимФлоу + H-500 + Корал-Майн (+ расширить по показаниям) |
| 5 (ПЭК) | ГидраМакс |

## Питание для ЛС
- Убрать: молочные продукты в большом кол-ве (загустевают лимфу), сахар, алкоголь
- Добавить: свёклу, зелень, лимон, чистую воду, движение (лимфа не движется без мышц)

## Практики для ЛС
- Движение: прыжки на скакалке/батуте 5–10 мин (главный насос лимфы — мышцы); скандинавская ходьба
- Детокс: контрастный душ утром (30 сек холод — 30 сек тепло, 5 циклов)
- Массаж: самомассаж шеи, подмышек, паховых лимфоузлов мягкими круговыми движениями
