## Программы по жалобам

**Частые ОРВИ / простуды**
→ "Быстрое восстановление при ОРВИ": Сильвер-Макс + КМД с Селеном + Колострум Плюс + Цинк Метионин 25

**Хронический кашель / бронхит**
→ "Очищение бронхов при хр. заболеваниях": Корал Солодка + КМД с мате + Колострум Плюс + АкваОкс

**Гайморит / синусит**
→ "От синуситов и гайморитов": Сильвер-Макс + КМД с мате + Колострум Плюс + ЛимФлоу

**Астма / аллергическое дыхание**
→ "Очищение лёгких при астме": Корал Босвелия + КМД с Селеном + Ассимилятор + Кальци-Прайм

**Курильщики**
→ "Очищение лёгких у курильщиков": Корал Солодка + ЛимФлоу + D-Спрей 2000 + Ассимилятор
