# ИС — Иммунная система

## Программы по жалобам

**Частые болезни / слабый иммунитет**
→ "Укрепление иммунитета у часто болеющих": Цинк Метионин 25 + Жир Печени Акулы + Алтимейт Макс + Колострум Плюс

**Аллергия**
→ "Против аллергии": D-Спрей 2000 + Супер-Флора + Ассимилятор + Омега-3 TG

**Грибковые инфекции**
→ "Противогрибковая программа": Метастик + Цинк Метионин 25 + Корал Люцерна + Липосомальный Куркумин

**Вирусные инфекции**
→ "Противовирусная программа": КМД с Селеном + Сильвер-Макс + Колострум Плюс + Жир Печени Акулы

**Аутоиммунные заболевания**
→ "При аутоиммунных заболеваниях": Корал Кошачий Коготь + Метастик + D-Спрей 2000 + Джинерра
⚠️ При аутоиммунных с вирусной природой (СКВ и др.) — НЕ стимулировать иммунитет агрессивно. Направление: противовирусное + антисептическое + нейропротекторное.

**Бактериальные инфекции**
→ "Противобактериальная программа": КМД с Селеном + Сильвер-Макс + Алтимейт Макс + D-Спрей 2000

## 5-уровневая эскалация для ИС

| Уровень | Продукты |
|---------|---------|
| 1 | Витамин D2000 |
| 2 | Витамин D2000 + Жир Печени Акулы |
| 3 | Витамин D2000 + Жир Печени Акулы + Алтимейт Макс |
| 4 | Витамин D2000 + Жир Печени Акулы + Алтимейт Макс + Pure-C |
| 5 (ПЭК) | Иммунити Пак |

## Важные ограничения
- Онкология (активная или ремиссия <2 лет): не рекомендовать иммуностимуляторы (Алтимейт Макс, Жир Печени Акулы, Колострум) без согласия онколога

## Питание для ИС
- Убрать: сахар (подавляет иммунитет на 6 часов), алкоголь, полуфабрикаты
- Добавить: цинк (тыквенные семечки, устрицы), витамин С (ягоды, перец, шиповник), чеснок, куркуму
