# ПС — Пищеварительная система

## Программы по жалобам

**Дисбактериоз / нарушение микрофлоры**
→ "Нормализация микрофлоры": Жевательная Папайя + Корал Бурдок Рут + Би-Курунга + Джинерра

**Запоры**
→ "Против запоров": Корал Магний + Каскара Саграда + Дейли Делишес Хай Файбер + Заферан

**Гастрит / слизистая желудка**
→ "Защита и восстановление слизистой желудка": Корал Солодка + Джинерра + Липосомальный Куркумин + Супер-Флора

**Печень / тяжесть в правом боку**
→ "Защита и восстановление печени": Лецитин + Заферан + ФанДетокс + АкваОкс

**Печень + паразиты**
→ "Очищение печени при паразитозах": Заферан + Лецитин Подсолнечный + ФанДетокс + ОписТоп

**Вздутие / дискомфорт в кишечнике**
→ "Оптимизация пищеварительной функции кишечника": Ассимилятор + Супер-Флора + Корал Люцерна + Заферан

**Паразиты / гельминты (взрослые)**
→ "Очищение кишечника от гельминтов": Корал Чёрный Орех + Экстракт Чеснока + Корал Артишок + Корал Магний

**Паразиты / гельминты (дети)**
→ "Детская паразитарная": Корал Чёрный Орех + Корал Магний + Би-Курунга + Экстракт Чеснока

**Желчный пузырь / застой желчи**
→ "Восстановление желчеоттока": Лецитин + Корал Артишок + Метастик + Заферан

**Хронические поносы**
→ "При хронических диареях": Жевательная Папайя + КМД с Селеном + Супер-Флора + Джинерра

## 5-уровневая эскалация для ПС

| Уровень | Продукты |
|---------|---------|
| 1 | Джинерра |
| 2 | Джинерра + Супер-Флора |
| 3 | Джинерра + Супер-Флора + Заферан |
| 4 | Джинерра + Супер-Флора + Заферан + ФанДетокс |
| 5 (ПЭК) | Здоровый Кишечник |

## Питание для ПС
- Убрать: фастфуд, жареное, молочный казеин при воспалении, газировку
- Добавить: клетчатку (овощи, зелень), ферментированное (квашеная капуста, кефир), воду до/после еды (не во время)

## Практики для ПС
- Движение: ходьба 15–20 мин после еды (стимулирует перистальтику)
- Привычки: жевать медленно (20+ жеваний); не есть на ходу и в стрессе
- Детокс: утром натощак — стакан тёплой воды с лимоном
