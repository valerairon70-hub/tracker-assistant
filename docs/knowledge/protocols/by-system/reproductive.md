# РС — Репродуктивная система

## Программы по жалобам

### Женские программы

**Нарушения цикла / ПМС**
→ "Восстановление менструального цикла 40+": Корал Магний + ФитоФормула для женщин + Омега-3 TG + Цинк Метионин 25

**Менопауза**
→ "Комфорт менопаузы": Кальци-Прайм + Либидэкстра для женщин + D-Спрей 2000 + Тейсти Би

**Подготовка к беременности**
→ "Подготовка к беременности": Пренатал+ + Коэнзим Q10 + D-Спрей 2000 + Спирулина Премиум

**Молочные железы / мастопатия**
→ "Здоровье молочных желёз": D-Спрей 2000 + Корал Келп + Липосомальный Куркумин + Корал Кошачий Коготь

### Мужские программы

**Потенция / либидо**
→ "Либидо мужчин": Либидэкстра для мужчин + Цинк Метионин 25 + Алтимейт Макс + Омега-3 TG

**Простата 55+**
→ "Здоровье предстательной железы 55+": Липосомальный Куркумин + Цинк Метионин 25 + Омега-3 TG + ОписТоп

**Тестостерон 50+**
→ "Восстановление тестостерона у мужчин 50+": Цинк Метионин 25 + Алтимейт Макс + D-Спрей 2000 + Противити Ультра

**Репродуктивное здоровье 30+**
→ "Восстановление репродуктивного здоровья у мужчин 30+": Цинк Метионин 25 + Кальци-Прайм + Корал Карнитин + Селен

## Особые программы

**Надпочечники / хронический стресс (связь с РС и ЭС)**
→ "Укрепление надпочечников": Тейсти Би + Противити Ультра + Сафрино + Корал Магний

**Тестостерон у мужчин**
→ Цинк Метионин 25 + Витамин D + Противити Ультра
Питание: цинк (тыква, устрицы), снизить пиво/алкоголь (эстрогеноподобные эффекты)

**Простата у мужчин**
→ Липосомальный Куркумин + Омега-3 TG
