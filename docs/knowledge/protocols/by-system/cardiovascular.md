# ССС — Сердечно-сосудистая система

## Программы по жалобам

**Высокое давление**
→ "Нормализация давления": Корал Таурин + АД Фито Контрол + Корал Магний + Лецитин

**Низкое давление**
→ "Скорая помощь при низком давлении": АД Фито Контрол + Кальци-Прайм + Гриффония + Коэнзим Q10

**Тахикардия / аритмия**
→ "Скорая помощь при тахикардии": Корал Таурин + ПентоКан + Коэнзим Q10 + Корал Магний

**Варикоз / тяжёлые ноги**
→ "Укрепление вен при варикозе": Алтимейт Макс + МСМ + ЦиркуФит + ФлексиКор

**Высокий холестерин**
→ "Снижение холестерина и профилактика атеросклероза": Корал Артишок + Ассимилятор + Корал Магний + Заферан

**Сгущение крови / тромбоз**
→ "Улучшение микроциркуляции и профилактика тромбозов": Омега-3 TG + ЦиркуФит + Экстракт Чеснока + Корал Магний

**Слабость сердечной мышцы**
→ "Клеточное питание для укрепления сердечной мышцы": Коэнзим Q10 + Корал Таурин + ПентоКан + Корал Магний

## 5-уровневая эскалация для ССС

| Уровень | Продукты |
|---------|---------|
| 1 | Коэнзим Q10 |
| 2 | ПентоКан + Коэнзим Q10 |
| 3 | ПентоКан + Коэнзим Q10 + Корал Магний |
| 4 | ПентоКан + Коэнзим Q10 + Корал Магний + ЦиркуФит |
| 5 (ПЭК) | КардиоПэк |

## Питание для ССС
- Убрать: соль >5 г/день, трансжиры, сладкое
- Добавить: жирную рыбу 2×/нед, калий (авокадо, бананы), магний, тёмный шоколад (умеренно)

## Практики для ССС
- Движение: ходьба 30 мин/день в умеренном темпе (не спринт); плавание
- Дыхание: 4–7–8 (вдох 4 сек, задержка 7, выдох 8) — нормализует давление
- Режим: не курить, контролировать вес; контрастный душ для сосудистого тонуса
