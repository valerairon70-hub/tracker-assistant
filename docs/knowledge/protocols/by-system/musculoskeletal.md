# КМС — Костно-мышечная система

## Программы по жалобам

**Набор мышечной массы / спорт / выносливость**
→ "Набор мышечной массы и восстановление": Противити Ультра + Ассимилятор + Корал Магний + Коэнзим Q10

**Суставы / скованность / хруст**
→ "Гибкость суставов, хрящей и связок": МСМ + ФлексиКор + Заферан + Оушенмин

**Воспаление суставов / артрит**
→ "Против артрита, снижение воспаления": Омега-3 TG + Корал Босвелия + D-Спрей 2000 + Цинк Метионин 25

**Остеопороз / хрупкость костей**
→ "Укрепление костей при остеопорозе": Кальци-Прайм + D-Спрей 2000 + Омега-3 TG + Оушенмин

**Коксартроз / тазобедренный**
→ "Программа при коксартрозах": МСМ + ФлексиКор + Корал Босвелия + Промарин Пептиды Коллагена

**Реабилитация после травмы**
→ "Реабилитация после травм": ФлексиКор + Цинк Метионин 25 + Ассимилятор + Кальци-Прайм

**Переломы / сращение костей**
→ "Программа при переломах": Кальци-Прайм + МСМ + ФлексиКор + Противити Ультра

| 1 | Кальци-Прайм |
| 2 | Кальци-Прайм + МСМ |
| 3 | Кальци-Прайм + МСМ + ФлексиКор |
| 4 | Кальци-Прайм + МСМ + ФлексиКор + Корал Босвелия |
| 5 (ПЭК) | Промарин (Пептиды Коллагена) |

## Питание для КМС
- Убрать: газировку (вымывает кальций), избыток кофеина, щавель и шпинат в большом кол-ве
- Добавить: кальций (кунжут, сардины с костями), витамин D, кремний (хвощ), белок

## Практики для КМС
- Движение: ходьба (не бег при артрозе), плавание, растяжка 10 мин утром
- Режим: не сидеть >1 часа без вставания; правильная осанка, ортопедическая обувь при нагрузках
- Детокс: тепловые процедуры (баня, сауна) — улучшают микроциркуляцию в хрящах
