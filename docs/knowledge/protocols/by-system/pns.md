# ПНС — Периферическая нервная система

## Программы по жалобам

**Остеохондроз / боли в позвоночнике**
→ "Восстановление при остеохондрозах": Би-Прайм Директ + МСМ + Промарин Пептиды Коллагена + D-Спрей 2000
(нервная проводимость — нужны активные формы B12 и B6; именно Би-Прайм Директ, не Тейсти Би)

**Снижение зрения**
→ "Восстановление зрения": Визи-Прайм + АкваОкс + Спирулина Премиум + Омега-3 TG

**Снижение слуха**
→ "Восстановление слуха": Тейсти Би + МайндСет + Омега-3 TG + D-Спрей 2000
