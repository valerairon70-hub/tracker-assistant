# МВС — Мочевыделительная система + Кожа

## Программы по жалобам

**Почки / частое мочеиспускание**
→ "Защита и восстановление функции почек": Корал Бурдок Рут + Корал Люцерна + Цинк Метионин 25 + Липосомальный Куркумин

**Мочевой пузырь / цистит / после воспаления**
→ "Восстановление мочевого пузыря после воспаления": Цинк Метионин 25 + Корал Люцерна + КМД с Селеном + Экстракт Чеснока

**Мочекаменная болезнь / подагра**
→ "Профилактика мочекаменной болезни": Оушенмин + ПентоКан + Корал Бурдок Рут + Тейсти Би

**Молодость кожи 40+**
→ "Сохранение молодости кожи 40+": Липосомальный Куркумин + МСМ + Цинк Метионин 25 + Селен

**Морщины / потеря упругости**
→ "Восстановление молодости кожи и уменьшение морщин": Промарин Пептиды Коллагена + Промарин Трипептиды Коллагена

**Проблемная кожа / прыщи**
→ "Очищение при проблемной коже": КМД с Селеном + Супер-Флора + МСМ + Липосомальный Куркумин

**Сухость / шелушение кожи 25+**
→ "Увлажнение и восстановление кожи при сухости": Липосомальный Куркумин + Промарин Пептиды Коллагена + Омега-3 TG + Жир Эму

**Выпадение волос**
→ "Укрепление волос": Спирулина Премиум + D-Спрей 2000 + МСМ + Корал Келп

**Ногти + шеллаки / ломкость**
→ "Здоровье ногтей при шеллаках": МСМ + Кальци-Прайм + Цинк Метионин 25 + Селен
