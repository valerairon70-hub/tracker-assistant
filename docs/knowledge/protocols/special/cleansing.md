# Программы очищения (Коло-Вада)

## Выбор программы

| Программа | Кому подходит | Цена (клубная) |
|-----------|---------------|----------------|
| Коло-Вада Лайт | Первое очищение, чувствительный кишечник, аллергики, без сои | ~6622 руб. |
| Коло-Вада Плюс | Глубокое очищение; ⚠️ содержит соевый лецитин — при аллергии на сою не подходит | ~7662 руб. |
| Коло-Вада Грин | Оптимальный выбор: эффективность Плюс + мягкость Лайт; без сои, без ГМО | ~8514 руб. |

---

## КОЛО-ВАДА ЛАЙТ (14 дней)

**3 этапа:**
1. Детоксикация — 7 дней
2. Очищение — 4 дня
3. Восстановление — 3 дня

**Состав:**
- Ди-Токс I (клевер, лопух, кукурузные рыльца, тимьян, эхинацея, имбирь, солодка)
- Ди-Токс II (имбирь, шалфей, лопух, артишок, расторопша, малина, алтей)
- ГринГрин
- Лайт Микс (псиллиум + семена льна + ягоды + хлорелла + бентонит)
- Ассимилятор
- Супер-Флора

Без сои, без ГМО.

---

## КОЛО-ВАДА ПЛЮС (Программа 2, 14 дней)

**Состав:** Алтимейт, Каскара Саграда, Корал Чёрный Орех, Набор Трав №2, Витамин С, Корал Люцерна, ДигестЭйбл, Супер-Флора, Коло-Вада Микс.

⚠️ Порошок Коло-Вада Микс содержит **соевый лецитин** — при аллергии на сою выбрать Коло-Вада Лайт или Грин.

---

## КОЛО-ВАДА ГРИН (14 дней)

Обновлённая версия Коло-Вада Плюс — на 2-м этапе 8 из 16 коктейлей заменены Лайт Миксом (мягче + фитонутриенты).
Без сои, без ГМО.
**Рекомендуемый вариант** по соотношению эффективности и мягкости.
