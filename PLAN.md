## Фаза 3.5 — Соответствие методу Бутаковой (на основе изучения материалов)

> Изучены материалы из папки Здоровье_Бутакова: скан книги 95 программ, зум-разборы диаграмм, концепция здоровья, 7 шагов метода, маршрутная карта.

### Что сейчас в инструменте НЕ соответствует методу:

1. **Нет уточнения состояния внутри системы** — метод содержит 95 конкретных программ.
   Например, для ЦНС: «бессонница», «антистресс для мужчин», «хроническая усталость», «депрессия», «головные боли» — это разные программы с разным набором продуктов.
   Сейчас Claude знает только «ЦНС в перегрузе» и подбирает всё подряд.

2. **Нет раздела «Коррекция питания»** — Шаг 4 метода Бутаковой. Каждый протокол включает рекомендации по питанию.

3. **Нет раздела «Дополнительные практики»** — Шаг 5: дыхание, ЛФК, детоксикация, психоэмоциональная коррекция.

4. **Пол клиента не используется для выбора программы** — поле есть, но протоколы для мужчин и женщин разные (особенно ЦНС, репродуктивная, иммунная).

5. **Возраст не используется** — для 60+ существуют отдельные программы (ЦНС 60+, память 60+).

6. **Нет концепции «Точка А → Точка Б»** — метод строится вокруг этого. В выводе должна быть цель через 90 дней.

### Что реализовать:

- [ ] **Уточняющий вопрос по главной системе** — после определения нагруженных систем добавить в форму поле «Основная жалоба» (выбор из списка по системе). Пример: ЦНС → [плохой сон / стресс / усталость / плохое настроение / головные боли]. Это позволит выбрать нужную из 95 программ.
- [ ] **Использовать пол для протокола** — в промпте явно прописать: если клиент мужчина → мужские программы, женщина → женские
- [ ] **Использовать возраст 60+** — если возраст ≥ 60, применять программы для этой возрастной группы
- [ ] **Добавить раздел «Коррекция питания»** в результат расшифровки (что исключить, что добавить под конкретные системы)
- [ ] **Добавить раздел «Дополнительные практики»** (дыхание, движение, детокс под конкретные системы)
- [ ] **Добавить «Точка А → Точка Б»** — формулировка текущего состояния и цели через 90 дней

### Что нужно от тебя для полной реализации:
- Содержимое 95 программ из книги Бутаковой (конкретные продукты на каждую программу) — скан есть, но OCR распознал только оглавление. Нужно либо вручную перенести ключевые программы, либо найти цифровую версию книги.
